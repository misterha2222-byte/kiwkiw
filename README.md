# Web Bucin untuk Pacar

Web statis sederhana yang menampilkan pesan romantis dengan background animasi gradient dan efek hati yang bergerak.

## Cara Menjalankan

1. Buka file `index.html` di browser web (Chrome, Firefox, dll.).
2. Web akan langsung menampilkan pesan romantis dengan animasi.

## Fitur

- Pesan romantis dengan background gradient yang beranimasi
- Efek hati yang bergerak naik turun
- Responsive design untuk semua device
- Tidak ada fitur edit - web statis murni

## Cara Upload ke GitHub/Cloudflare

1. **Buat repository baru di GitHub**
2. **Upload semua file** dari folder ini:
   - `.gitignore`
   - `.nojekyll`
   - `README.md`
   - `index.html`
   - `static/` (folder lengkap)
3. **Aktifkan GitHub Pages** atau **connect ke Cloudflare Pages**
4. **Selesai!** Web langsung bisa diakses

## Teknologi

- HTML5
- CSS3 (dengan animasi gradient)
- JavaScript (hanya untuk animasi hati)
